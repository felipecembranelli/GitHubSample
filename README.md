# GitHubSample

Sample ASP.NET MVC application using Github API. It allows to list user repositories, search GitHub repos and mark as favorite.

Technologies/Features:
- ASP.NET MVC
- Entity Framework 6
- .NET 4.5
- Depency Injection with  AutoFac
- Unit Testing with Moq framework
